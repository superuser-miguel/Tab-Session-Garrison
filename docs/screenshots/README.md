# Screenshots

README images live here. Drop PNGs in this folder and reference them from the
top-level `README.md` with relative paths, e.g. `docs/screenshots/popup.png`.

Filenames the README currently expects (add/rename as you like, then update the
README to match):

- `popup.png` — main view: session list + detail pane (hero image)
- `multi-select.png` — several sessions selected + the "N selected" panel
- `tab-groups.png` — tab-group chips (detail pane) and dots (list)

Committed images are version-controlled and travel with the repo. Keep them
reasonably sized (a few hundred KB each; downscale huge captures).
